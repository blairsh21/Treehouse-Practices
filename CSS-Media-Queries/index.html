Let’s Talk Media Queries.

You know what makes CSS even better? Media queries. If you’re not familiar with CSS media queries, you should be. Media queries allow developers to apply styles based on the type of device viewing the web page, such as print versus screen, or based on specific characteristics such as viewport width.

Just imagine, before CSS media queries were introduced, it was well-nigh impossible to create a great online experience on a variety of devices like phones, tablets, laptops, and desktop computers. But these days, CSS media queries make cross-device wizardry a breeze!
